# CS50 Game Development

This repository contains my exercises and projects from the **CS50 Introduction to Game Development** course.

The purpose of this repository is to practice and document the concepts taught in the course, including game loops, rendering, input handling, and basic game mechanics.

## Technologies

The projects in this repository are built using:

* Lua
* LÖVE (Love2D)

## Content

Each directory corresponds to a lecture or project from the course and contains the code developed while following the course materials.

The code here focuses on learning and experimenting with the techniques demonstrated in the course.

## Note

This repository is intended for educational purposes and personal practice.
